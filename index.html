MWGL Studio v2

MWGL v2：节点含 start（唯一入口）、wait_user（等待用户交互，中间节点）、trigger（与 switch 配套的触发条件）、switch / loop（出边须 label）、parallel（并行分支）、case、success / failure（终态无出边）。图保持 DAG；迭代用 loop 的「继续/退出」分支或 case 内 【循环】 说明，勿用连线回环。紧凑文本支持 RULE、VERSION 2、开始、条件、CASE；复杂图用 MODE graph + NODE/EDGE。

工作流默认居中在画布内；空白处拖拽平移；Ctrl+滚轮缩放；节点可拖动；Shift+拖节点可连线；点击边可选中并按 Delete/Backspace 删除。
没有选中节点时，将自动选择第一个节点。
